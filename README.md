# AfricAI
AI-powered learning platform where tutors host and post lessons, and learners access them at a flat rate. Designed to combine education with smart, interactive AI support.

AfricAI

**AfricAI** is an AI-powered learning platform designed to connect tutors and learners in one place.  
Tutors can host and post lessons, while learners can access any lesson of their choice at a flat rate.  
The platform integrates AI to enhance the learning experience, making education more interactive, adaptive, and accessible.  

---

## 🌍 Vision  
To create a modern, AI-driven education hub where high-quality learning is affordable, personalized, and available to everyone.  

---

## ✨ Key Features (Planned)  
- 📚 **Tutor Dashboard** – Tutors can create, upload, and manage lessons.  
- 🎓 **Learner Access** – Learners can pick any lesson for a flat rate of R2000.  
- 🤖 **AI Integration** – AI tutoring, interactive Q&A, and personalized recommendations.  
- 🔒 **Secure Payments** – Simple, safe payment system for all transactions.  
- 📈 **Scalability** – Built to expand across subjects, regions, and levels.  

---

## 🛠️ Tech Stack (Planned)  
- **Frontend:** HTML, CSS, JavaScript (React optional)  
- **Backend:** Java   
- **Database:** MongoDB  
- **AI Integration:** OpenAI / Gemini / Deepseek 
- **Hosting:** To be decided (AWS, Azure, or similar)
